Help

Play this Simon game and have fun testing your memory. Simon lights one of the coloured pads and sounds a tone, then two, then three, etc. Player attempts to match Simon by pressing the pads in the proper sequence If player fails he loses a life, if lives run out - the game is over. Game HARD MODE has 20 rounds, that means that in the last round player needs to repeat 20 signals, and only 2 lives - player has only one chance for a mistake. There are 10 rounds and 3 lives to lose in the EASY MODE. Rack up high scores, improve your memory and enjoy this classic game.

Good Luck!
