Test your logic and learn more about RGB values with this RGB color challange.

You have 2 attemps to guess the right color of suggested RGB combination.

Find and select the square whose color is described by the given RGB value.

Note: the three numbers in an RGB color range from 0 to 255 and describe the amount of red, green, and blue, respectively.

For example, rgb(255, 0, 0) is pure red, rgb(0, 0, 255) is pure blue, and rgb(255, 0, 255) is magenta.

Pure black is rgb(0, 0, 0,) and pure white is rgb(255, 255, 255).

Remember, if you make a mistake twice - the game will be over.

Select EASY game mode to play with 3 colors and HARD mode to play with 6 colors.

Select NEW COLORS to start a new game.

Good Luck!
